Tourists

Does the number of tourists influence the number of Oktoberfest visitors?

In this visualisation we would show up how the number of tourists in the city of munich influence the number of Oktoberfest visitors. In the bar chart we can see, that the number of visitors fall up to 5.600.000 visitors in 2016. Contrary to this, the number of tourists tends up in month september each year.

Unfortunately, the dataset for the tourists in munich starts from 2006 to 2016, and is parted in months. So we decided to use the values in september to represent this visualisation, because the Oktoberfest starts in middle of september and ends up in frist october week.
From the dataset we chosse the values for guests and overnight stays in munich. This incudes the guests from inland and foreign countries.

In the bar chart we can select the dataset we would see, so we can focus in the visualisation.
When we focus in guests and overnight stays, we can see that there are much more people in the city (quite probable for Oktoberfest) then the years before.
